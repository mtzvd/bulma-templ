## Regenerating Documentation

Documentation is auto-generated from source code comments using the `docgen` tool.

**Generate all docs:**
```bash

```

**Generate for specific package:**
```bash

go run cmd/docgen/main.go -package components
go run cmd/docgen/main.go -package form
go run cmd/docgen/main.go -package layout
go run cmd/docgen/main.go -package columns
go run cmd/docgen/main.go -package grid
```

**Generate for single file:**
```bash
go run cmd/docgen/main.go -file elements/button.templ
```
